# ①課題名
英語学習アンケート集計アプリ

## ②課題内容（どんな作品か）
- 以前作成した英単語学習アプリの改善に向けた事前アンケートとして位置づけた、アンケート入力・集計・表示を行うPHPアプリです。

## ③アプリのデプロイURL
- http://yone-ken.sakura.ne.jp/yonemoto_19_php01/post.php

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: 今回なし
- PW: 今回なし

## ⑤工夫した点・こだわった点
- ユーザーの英語レベル、アプリ満足度、欲しい機能、自由コメントを収集し、改善方針検討のためのデータを可視化
- post.php / write.php / read.php を分離し、入力・保存・表示の役割を明確化
- CSVファイルでアンケート結果を管理
- アンケート結果を全件削除できるリセット機能を実装（誤操作防止の confirm ダイアログ付き）

## ⑥難しかった点・次回トライしたいこと（又は機能）
- CSVファイルの文字コードと、Excelでの文字化け対応
- ファイル操作（追記・リセット）と表示処理の設計整理

- 満足度の平均値や件数などの簡易集計機能
- 英単語アプリ本体とアンケート結果の連携（改善内容をアプリに反映する仕組み）
